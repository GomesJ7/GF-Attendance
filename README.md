------ Mise en place du Projet -------

1 - Clone le projet
2 - faire un ```composer i ```  pour réccupérer toutes les dependancies
3 - Ajouter le .env
4 - charger les fixtures
```symfony console doctrine:fixtures:load```
5 - Lancer le projet
start server
symfony server:start -d

navigate to server
start https://127.0.0.1:8000

stop server
symfony server:stop
